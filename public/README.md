# Hướng dẫn: Copy file logo.png vào thư mục này

Thư mục này chứa các file tĩnh (static assets) như:
- logo.png (logo của app)
- favicon.ico
- images, fonts, etc.

Tất cả file trong thư mục này sẽ được serve tại root URL.

Ví dụ: 
- public/logo.png → http://localhost:3000/logo.png
- public/favicon.ico → http://localhost:3000/favicon.ico
